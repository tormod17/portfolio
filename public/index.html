Harness Talent

Automated Recruitment Site

New

The stack used to build this site was node.js, redis, aws S3, semantic U.I and handlebars templating engine. We also used Linkedin 0Auth2, and mailgun with ViewerJS and a web app viewer from microsoft. For testing we used Travis for C.I. and for coverage istanbul, codecov was used for our online reports

The site was geared towards laptop, and gives different users permissions which allow them access to certain parts of the site. Storing credentials in a cookie generated and encrypted by Hapi Auth Cookie. All CV's are stored in the cloud using AWS S3 these are uploaded them from the front end.
